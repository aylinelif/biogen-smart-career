# 🚀 BioGen AI - Akıllı Kariyer Asistanı

BioGen AI, kullanıcıların dağınık kariyer notlarını, deneyimlerini ve yeteneklerini saniyeler içinde profesyonel bir **LinkedIn "Hakkında"** yazısına dönüştüren yapay zeka destekli bir Micro-SaaS prototipidir.

## ✨ Öne Çıkan Özellikler
- **Google Gemini API Entegrasyonu:** En güncel yapay zeka modelleriyle yüksek kaliteli metin üretimi.
- **Dinamik Model Tarama (Model Discovery):** Kullanıcı API yetkisine göre en uygun modeli (`Gemini 1.5`, `2.0` vb.) otomatik tespit eden esnek mimari.
- **Modern UI/UX:** Tailwind CSS ve Plus Jakarta Sans fontu ile şık ve kullanıcı dostu tasarım.
- **Hızlı Kopyalama:** Üretilen metni tek tıkla panoya kopyalama özelliği.

## 🛠 Kullanılan Teknolojiler
- **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla JS)
- **AI:** Google Gemini API
- **Deployment:** Netlify
- **Versiyon Kontrol:** Git & GitHub

## 🚀 Canlı Uygulama
Projeyi canlıda test etmek için: [https://biogen-smart-career.netlify.app/](https://biogen-smart-career.netlify.app/)
